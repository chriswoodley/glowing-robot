# Layout Column

A `.layout_column` is a container with either an implicit or explicit width containing your content.
By default `.layout_column` will be stacked for mobile within a `.layout_row`.

## Equal Width Columns

When applying `.layout_row--{sm, md, lg, xl}`...`.layout_column` will implicitly allocate width based
on remaining space of the `.layout_row` for small screens and up.

## Content Width Columns

`.layout_column .layout_column--sm-auto` will make the width of the column
the same width as the content inside it for small screens and up. You can also
control when this behavior happens for `md`, `lg`, or `xl` screen sizes.

## Column Grid Widths

`.layout_column .layout_column--sm-1` represents 1 out 12 column widths starting at
small screens and up. You can control when to display column width for `md`, `lg`, or `xl`
screen sizes.

To make a column span across multiple columns by using the column number. For example,
`.layout_column .layout_column--sm-4` will result in a column that spans over 4 columns for
small screens and up.

A simple 2 column layout may look something like:

<Unstyled>
  <div className="layout_row layout_row--sm">
    <div className="layout_column layout_column--sm-4">
      <div className="box">aside content</div>
    </div>
    <div className="layout_column layout_column--sm-8">
      <div className="box">main content</div>
    </div>
  </div>
</Unstyled>

<Source
  code={`
    <div class="layout_row layout_row--sm">
      <div class="layout_column layout_column--sm-4">
        <div class="box">aside content</div>
      </div>
      <div class="layout_column layout_column--sm-8">
        <div class="box">main content</div>
      </div>
    </div>
  `}
  language="html"
  dark
  format
/>

Notice that the total column numbers `4 + 8 = 12`. You can combine various column configurations
as long as it totals to 12 columns.

# Column Order

We can order the column from 1 to 12. For instance we may need to display content in a different
order based on screen size. For example:

`.layout_column .layout_column--sm-3 .layout_column--sm-order-4`. Keep in mind that **you will need
to apply order modifiers to all columns** to get the desired order of columns.

<Unstyled>
  <div className="layout_row">
    <div className="layout_column layout_column--sm-4 layout_column--sm-order-3">
      <div className="box">1</div>
    </div>
    <div className="layout_column layout_column--sm-4 layout_column--sm-order-2">
      <div className="box">2</div>
    </div>
    <div className="layout_column layout_column--sm-4 layout_column--sm-order-1">
      <div className="box">3</div>
    </div>
  </div>
</Unstyled>

## Column Offset

Column offset `.layout_column .layout_column--sm-4 .layout_column--sm-offset-4` will offset the
column by 4 grid column units by adding a left margin to it.
