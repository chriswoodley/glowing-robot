# Flex

`.flex` is a utility css class to apply `display: flex` to your element. It can be applied to
mobile, `sm`, `md`, `lg`, and `xl` screens. The example below applies `display: flex` to
medium screen size and up.

<Unstyled>
  <div className="flex--md">
    <div className="box">1</div>
    <div className="box">2</div>
    <div className="box">3</div>
  </div>
</Unstyled>

## Flex Direction

The `.flex--row`, `.flex--column`, `.flex--row-reverse`, and `.flex--column-reverse` sets
how `.flex_item` elements are placed in the `.flex` container defining the main axis and the
direction (normal or reversed). By default, flex items will be laid out in a row.

It can be applied to mobile, `sm`, `md`, `lg`, and `xl` screens.

<Unstyled>
  <div className="flex--md flex--lg-row-reverse">
    <div className="box">1</div>
    <div className="box">2</div>
    <div className="box">3</div>
  </div>
</Unstyled>

## Flex Wrapping

The `.flex--wrap`, `.flex--wrap-reverse`, and `.flex--nowrap` sets whether flex items are forced
onto one line or can wrap onto multiple lines. If wrapping is allowed, it sets the direction
that lines are stacked. By default `.flex` wrapping is not allowed.

It can be applied to mobile, `sm`, `md`, `lg`, and `xl` screens.

<Unstyled>
  <div className="flex flex--wrap">
    <div className="box" style={{ width: "40%" }}>
      1
    </div>
    <div className="box" style={{ width: "40%" }}>
      2
    </div>
    <div className="box" style={{ width: "40%" }}>
      3
    </div>
  </div>
</Unstyled>

## Flex Alignments

To align our `.flex_item` we use the `.flex--align-items-flex-start`, `.flex--align-items-center`,
`.flex--align-items-flex-end`, and `.flex--align-items-stretch` to align our item on the
vertical axis.

We use `.flex--justify-content-flex-start`,`.flex--justify-content-flex-end`, and
`.flex--justify-content-center` to align `.flex_item` on the horizontal axis.

It can be applied to mobile, `sm`, `md`, `lg`, and `xl` screens.

<Unstyled>

    <div className="box">2</div>
    <div className="box">3</div>
  </div>

# Flex Spacing

The `.flex--justify-content-space-between`, `.flex--align-content-space-between` spaces `.flex_item`
evenly distributed within the alignment container along the main axis. The spacing between each pair
of adjacent `.flex_item` is the same. The first item is flush with the main-start edge, and the
last item is flush with the main-end edge.

It can be applied to mobile, `sm`, `md`, `lg`, and `xl` screens.

<Unstyled>
  <div className="flex flex--md-justify-content-space-between">
    <div className="box">1</div>
    <div className="box">2</div>
    <div className="box">3</div>
  </div>
</Unstyled>

`.flex--justify-content-space-around` and `.flex--align-content-space-around` the items are evenly
distributed within the alignment container along the main axis. The spacing between each pair of
adjacent items is the same. The empty space before the first and after the last item equals half
of the space between each pair of adjacent items.

It can be applied to mobile, `sm`, `md`, `lg`, and `xl` screens.

`.flex--justify-content-space-evenly` and `.flex--align-content-space-evenly` the items are evenly
distributed within the alignment container along the main axis. The spacing between each pair of
adjacent items, the main-start edge and the first item, and the main-end edge and the last item,
are all exactly the same.

It can be applied to mobile, `sm`, `md`, `lg`, and `xl` screens.
