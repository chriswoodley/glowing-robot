# Lazy Cover Image

A `<LazyCoverImage>` lazily loads and displays an image that will cover its parent container. 
It is meant to be used with `<Suspense>`. Optionally, it can be used with `<AspectRatioContainer>`.

<Canvas>
  <Story
    name="Default"
    args={{

      alt: 'my bp station'
    }}

    {(args) => <LazyCoverImage {...args} />}
