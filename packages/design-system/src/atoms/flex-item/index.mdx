# Flex Item

`.flex_item` is a utility css class to apply flex properties to your flex item controlling flex
basis and order. `.flex_item` should be a child of a `.flex` container. A `.flex_item` layout 
behavior can be modified with the following classes: `.flex_item--grow-0`, `.flex_item--grow-1`,
`.flex_item--shrink-0`, `.flex_item--shrink-1`, `.flex_item--none`, `.flex_item--basis-auto`,
`.flex_item--basis-{1-12}`, `.flex_item--order-first`, `.flex_item--order-last`,
`.flex_item--order-{1-12}`. These modifiers can be applied to mobile screens,
`sm`, `md`, `lg`, and `xl` screens.

Each `.flex_item` will grow and shink automatically to fill the remaining space. This is the default
behavior.
