# Headings

The <strong><code>&lt;h1&gt;</code></strong> to <strong><code>&lt;h6&gt;</code></strong> 
<a href="/en-US/docs/Web/HTML">HTML</a> elements represent six levels of section headings. 
<code>&lt;h1&gt;</code> is the highest section level and <code>&lt;h6&gt;</code> is the lowest.
